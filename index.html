SQL命令速查

Posted on 2021-03-15

WHERE

提取满足指定条件的记录

  • operator: =, >, <, >= , <=, !=, <> 表示（不等于），可使用 not, and, or 连接
  • is null: 判空
  • between A and B: [A, B] 之间的值
  • in (a, b, c): 等于 a 或 b 或 c 的值
  • like 模糊查询
    • % 表示多个字值，**_** 下划线表示一个字符
    • M% : 查询以 M 开头的内容
    • %M% : 查询包含 M 的所有内容
    • %M_ : 查询 M 在倒数第二位的所有内容

ORDER BY

对结果集进行排序

  • 默认升序

LIMIT

限制 SELECT 返回的数据数量

JOIN

  • INNER JOIN：如果表中有至少一个匹配，则返回行
  • LEFT JOIN：即使右表中没有匹配，也从左表返回所有的行
  • RIGHT JOIN：即使左表中没有匹配，也从右表返回所有的行
  • FULL JOIN：只要其中一个表中存在匹配，则返回行
  • 通过 A.colname 可以访问到 A 表里的 colname 列

函数

MAX()

返回指定列的最大值

  • 有多个最大值会返回多个最大值

模糊图像

Posted on 2021-03-11

.tif 为扩展名的文件是 TIFF 文件。

高斯模糊

表面模糊

  • 总体来说，表面模糊比高斯模糊更精细
  • 表面模糊中的半径是指模糊取样区域的大小。
  • 表面模糊中的阈值，是控制相邻像素色调值与中心像素值相差多大时才能成为模糊的一部分。色调值差小于阈值的像素被排除在模糊之外。

动感模糊

  • 动感模糊一般用来模拟相机高速跟拍的效果
  • 角度决定了模糊的方向，0为水平方向；90为垂直方向
  • 距离决定了模糊程度，可以使用滑块调整，也可直接输入数值

224. 基本计算器

Fine-grained Video-Text Retrieval with Hierarchical Graph Reasoning

Posted on 2021-03-10 Edited on 2021-03-15

摘要

通过学习联合嵌入空间来度量跨模态相似度的方法，不能很好的表示诸如：场景、目标、动作及其构成等，复杂的视觉和文本细节。本文提出 HGR （Hierarchical Graph Reasoning，分层图推理）模型。HGR 将图像-文本匹配分解为从全局到局部的多个层次（global-to-local levels）。具体来讲，HGR 将文本结构成层次化语义图，包括：事件（events）、动作（action）、实体（entities）以及各层间的关系。基于注意力机制的图推理被用来生成分层文本嵌入，从而指导学习各种不同层次的视频学习。
